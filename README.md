# Sales Data EDA 📊

Exploratory Data Analysis on retail sales data using Python.

## 🔍 Key Analysis
- Gender & Marital Status distribution
- Zone-wise Orders (Pie chart)
- Top 10 States by Amount
- Age Group vs Zone breakdown
- Outlier detection using IQR & Box plots

## 🛠 Libraries Used
- Pandas, NumPy, Matplotlib, Seaborn.
